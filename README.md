# Bite Right
This is a short, interactive browser-based game designed to make learning about nutrition fun and engaging. The experience uses simple, game-style mechanics to help users understand how everyday food choices can impact energy, focus, and overall well-being.

The game simulates real-life food decisions through a series of mini-games, where users learn by playing — not through traditional quizzes or lectures.

Built as a microlearning tool, the experience takes approximately 3–5 minutes to complete and combines:

Fast-paced arcade-style gameplay (slice, match, build, choose)
Immediate feedback and bite-sized nutrition tips
Visual and interactive learning (no long text or lectures)
A scoring system and leaderboard to encourage replayability

The game is designed for general audiences and can be used for wellness campaigns, educational content, or as a light, engaging introduction to better nutrition habits.
